subroutine setup_decay_chain ( integer, intent(in)  ic)

Sets up one decay chain.

Reads files NUCID.chain to get nuclides in the chain, then reads ENSDF decay data files for each nuclide to get decay data.

Note
Could be made more automated: get_requested_decay_chains could read top parent and stable daughter nuclides for each chain, then setup_decay_chain would go from the stable daughter up using ENSDF info (rather than reading in the NUCID.chain file with all nuclides involved).
