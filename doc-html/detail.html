Detailed Description

Publication

O. Šrámek, L. Stevens, W. F. McDonough, S. Mukhopadhyay, and J. Peterson: “Subterranean production of neutrons, 39Ar and 21Ne: Rates and uncertainties.” Submitted to Geochim. Cosmochim. Acta (arXiv:1509.07436).

Overview

Software argon39 handles the calculation of nucleogenic production rates of neutrons, 21Ne and 39Ar in underground geological settings. Prepares input files for calculations of cross sections and spectra using TALYS code. Prepares input files for neutron history simulations using MCNP6 code and reads results from its output files.

  • Alpha particles are produced in decay chains of long-lived natural radionuclides (mainly 232Th and 238U).
  • Neutrons are produced by (α,n) reactions on light nuclides and by spontaneous fission of 238U.
  • 21Ne is produced by (α,n) on 18O.
  • 39Ar is produced by (n,p) reaction on 39K.

Four programs

Contains 4 separate programs; the corresponding executables are

  • argon39.x
  • talysprep.x
  • mcnp6extract.x
  • mcnp6prep.x

Basic calculation sequence

0a. (optional) run talysprep.x to prepare input files for TALYS.

0b. (optional) Run TALYS to calculate (α,n) cross sections and spectra.

1. Run argon39.x to calculate α production rates and spectra, α range, neutron production functions, neutron yields and production rates and spectra

2. Run mcnp6prep.x to prepare input files for MCNP6.

3. Run MCNP6 simulations.

4. Run mcnp6extract.x to extract results from MCNP6 output files.

As TALYS-calculated (α,n) cross sections and spectra are included in the distribution, steps 0a and 0b are optional.

argon39.x

Description

argon39.x calculates natural α particle emission and neutron production. Uses (α,n) cross sections and spectra calculated by TALYS. List of the main subroutines with brief description follows:

  • get_inputs_start_output
  • read_elements_nuclides ... Reads in atomic & nuclide mass data and nuclide natural abundances (data from NIST)
  • read_abundances ... Reads in rock composition (see publication for references)
  • setup_decay_chains ... Sets up requested decay chains by reading in parameters from ENSDF decay datasets
  • read_stopping_power ... Reads in stopping power calculated externally (e.g., by SRIM)
  • calculate_alpha_range
  • output_possible_targets
  • get_requested_an_targets
  • setup_target_nuclide
  • read_an_cross_section ... Reads in (α,n) cross section, calculated externally (e.g., by TALYS), for requested target nuclides
  • calculate_an_neutron_yield ... Calculates (α,n) neutron yield
  • calculate_an_neutron_spectrum ... Calculates neutron energy spectra
  • calculate_SF_neutron_production
  • finalize_output

Execution

In subdirectory example/, run 

../argon39.x argon39_example.inp

Input file

The name of the input file (e.g., argon39_example.inp) is the one required command line parameter. The input file contains the following inputs: 

model_name = "example",

Main output file argon39_MODELNAME.out is created upon execution. 

input_directory = "../inputdata",

Directory where input data are located. All following input files are assumed to reside in this directory. 

file_abundances = "elem_wtfrac_RG03UC+C",

Name of 2-column input file with elemental rock composition:

  1. Atomic number (Z)
  2. Weight fraction of element
file_atomic_mass = "atom_wt_nist_full.94",

Name of a 6-column input file with atomic masses:

  1. Atomic number (Z)
  2. Atomic symbol
  3. Mass number (A)
  4. Relative nuclide mass in u (= g/mol)
  5. Natural isotopic composition (isotopic fraction)
  6. Standard atomic weight in u (= g/mol)

If isotopic composition is unknown, put zero in the input file. If standard weight is unknown, put mass number. 

file_decay_chains = "decay.chains",

Name of input file with NUCIDs of top parent in each decay chain to be included (e.g., 232TH), one NUCID per line. 

file_stopping_power = "srim_tot_RG03UC_14ThU",

Name of a 2-column input file with mass stopping power of α particle in the rock:

  1. Energy of α particle in MeV
  2. Mass stopping power in MeV.cm2/g
file_an_targets = "targets_tendl-2012.an",

Name of input file with NUCIDs of (α,n) target nuclides (e.g., 27Al), one per line. 

which_an_calc = 2,    !! 1=local(kinematic), 2=talys
local_directory = "../localxs",
talys_directory = "../talysxs",

Leave which_an_calc = 2. talys_directory is the name of the directory which contains the structure of subdirectories with (α,n) cross sections and spectra calculated by TALYS for the desired target nuclides. These cross sections/spectra are included in this distribution, but may also be calculated anew (see talysprep.x). 

rock_density = 2.70,    !CRUST2.0 average density in layers 3-5

Material density of the rock in g/cm3. 

dEan = 0.01,

Desired energy stepping (in MeV) of neutron production calculations. 

dEtalys = 0.1,

Energy stepping (in MeV) of TALYS cross sections, relevant for talysprep.x. 

mcnp6_accu = 0.005,
mcnp6_radcm = 300.,

Accuracy criterion for MCNP6 simulations and the radius (in cm) of spherical domain of the MCNP6 simulations. Relevant for mcnp6prep.x and mcnp6extract.x.

Output

  • argon39_MODELNAME.out ... main output file, description of what has been done.
  • alpha_energy_spectrum_PPP.out ... spectrum of α particles from PPP (e.g., 232TH) decay chains
  • alpha_energy_table_PPP.txt ... table of α particle emission from PPP decay chains
  • alpha_range.out ... range of α particles
  • neutron_prod_func_TTT.out ... thick target neutron production function for target nuclide TTT (e.g.. 27AL)
  • neutron_spectrum_232TH_13C.out ... (α,n) neutron spectrum with αs from PPP chain and TTT target
  • RESULTS_nrate_yr-kg-wtf-wtf.out ... table of (α,n) and spontaneous fission neutron production rate per year per kg of rock per wt.frac. of PPP element (e.g., thorium) per wt.frac. of TTT element (e.g., aluminum)
  • RESULTS_nrate_yr-kg.out ... table of (α,n) and spontaneous fission neutron production rate per year per kg of rock
  • RESULTS_nyield_per_decay.out ... table of (α,n) and spontaneous fission neutron yield
  • sdef_si_PPP_TTT.mcnp6 and sdef_sp_PPP_TTT.mcnp6 ... neutron spectrum input files for MCNP6 simulation
  • zaid_atomfrac_example.mcnp6 ... input file for MCNP6 simulation, nuclide atomic fractions in the rock

talysprep.x

Description

talysprep.x can be used to prepare the calculation of (α,n) cross sections and spectra by TALYS, if desired and if TALYS is installed on your machine.

Uses same Input file as argon39.x.

Execution

In subdirectory example/, run 

../talysprep.x argon39_example.inp

Output

  • TALYSprep.out ... main output file with basic information.
  • TALYSbatch_example.sh ... bash script to run TALYS calculations.

mcnp6prep.x

Description

mcnp6prep.x prepares input files for MCNP6 simulations.

Uses same Input file as argon39.x.

Execution

In subdirectory example/, run 

../mcnp6prep.x argon39_example.inp

The program interactively asks for some values. You can just enter 0s the first time. Then run MCNP6 in the "initiate" mode (mcnp6 i name=inputfile), then inspect the output file (input file name with appended o) to calculate input quantities asked for during the second run of mcnp6prep.x. _[Of course, this could be automated too...]_

Output

  • MCNP6batch_example.sh ... ... bash script to run TALYS calculations.
  • mcnp6_MODELNAME_PPP_TTT ... series of input files for MCNP6, where PPP is NUCID of decay chain parent and TTT is NUCID of (α,n) target, e.g., mcnp6_example_232TH_27AL.

mcnp6extract.x

Description

Extracts results from MCNP6 output files, after MCNP simulations have been run.

Uses same Input file as argon39.x.

Execution

In subdirectory example/, run 

../mcnp6extract.x argon39_example.inp

Output

  • MCNP6extract_MODELNAME_tally0.out ... tally of neutron flux out of the calculation domain; should be all 0s.
  • MCNP6extract_MODELNAME_tally1.out ... tally of (n,p) 39Ar yield per neutron per wt.frac. K (first 3 columns) and corresponding statistical uncertainty (next 3 columns)
  • MCNP6extract_MODELNAME_tally2.out ... tally of (n,α) 21Ne yield per neutron per wt.frac. Mg (first 3 columns) and corresponding statistical uncertainty (next 3 columns)
