double precision  a,

This routine computes the nth stage of refinement of an extended trapezoidal rule.

func is input as the name of the function to be integrated between limits a and b, also input. When called with n=1, the routine returns as s the crudest estimate of int(f(x),x=a..b) Subsequent calls with n=2,3,... (in that sequential order) will improve the accuracy of s by adding 2^(n-2) additional interior points. s should not be modified between sequential calls.

Author
W.H. Press, S.A. Teukolsky, W.T. Vetterling, B.P. Flannery: Numerical Recipes in Fortran 77
